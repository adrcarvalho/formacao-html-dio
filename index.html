A estrutura básica do HTML

HTML significa Hyper Text Markup Language. Trata-se, portanto, de uma linguagem de marcação, e não programação. Dessa forma, o HTML serve para você indicar para o navegador o que é cada coisa dentro de uma página: um texto, uma tabela; se esse texto está em negrito, itálico etc. E muito mais...

Iniciamos a criação de um arquivo HTML criando um arquivo chamado "index.html". Nele, inserimos a primeira TAG "html:5", que irá preencher as primeiras informações essenciais para a criação de um site. Nessa tag encontramos o idioma da página, título etc.

Um documento HTML deve começar e terminar com a tag "html". Após essa tag, inserimos a tag "head", que contém meta informações bem como o título do site (aquele que aparece na aba do navegador). Após, temos a tag "body". TUDO que estiver dentro da tag BODY será visível pelo navegador.

Ao iniciar o estudo de uma linguagem é interessante também ler a documentação dela, pois é lá que todas as regras e funcionalidades estarão dispostas. No caso do html existe um site para isso.

Tags sem fechamento

Algumas tags não possuem fechamento, como é o caso da tag "input". Com ela você cria um campo que pode ser inserido algum texto ou número, dependendo do type inserido, isto é, "text" ou "number".

Outra tag que não possui fechamento é a tag de imagem < img > e a tag < br >.
